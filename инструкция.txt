1) Данные выводятся с помощью таблицы, ключ с которого берутся данные называется data, который находится обьекте const data (const data -> data);
2) Для наименование таблицы используется ключ header, который находится в обьекте const data (const data -> header);
3) Для добавления дополнительных классов к таблице используется ключ cssClass, который находится в обьекте const data, в ключе cssClass, находятся доп ключи для формирования стилей заголовка и таблице (const data -> cssClass -> header || table);
4) Для наименование колонок используется ключ attribute (const data -> attribute);
5) Для отрисовки таблицы и заголовка используется методы render() и drawHeading(), они находятся в классе GridView
